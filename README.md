# Abdul Rehman - Portfolio Website

Senior Flutter & Full-Stack Developer with a lot of projects and working websites. "Elegance is my North-Star". I Build What You Get Amazed OF!

This is a modern and responsive portfolio website built with Next.js and styled using Tailwind CSS to showcase my projects, skills, and experiences.

## Features

- **Responsive Design**: Optimized for all devices, including mobile, tablet, and desktop.
- **Dark Theme**: Provides a visually appealing dark theme.
- **Smooth Animations**: Utilizes Framer Motion for animations.
- **Dynamic Content**: Powered by a configuration file for easy updates.

## Technologies Used

- **Frontend**: Next.js, React, HTML, CSS, Tailwind CSS
- **Mobile**: Flutter, Dart
- **Backend**: Node.js, Python, C++, C#
- **Tools**: Android Studio, VS Code, WordPress
- **Animation**: Framer Motion

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/itxabdulrehman1/Portfolio.git
   ```

2. Navigate into the project directory:

   ```bash
   cd portfolio-website
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
